# Thai-Chinese Seafood Restaurant - Design System

## Design Philosophy

### Visual Language
- **Cultural Fusion**: Seamless blend of Thai elegance and Chinese sophistication
- **Premium Quality**: Luxury dining experience with authentic cultural elements
- **Mobile-First**: App-like interface optimized for iPhone and mobile devices
- **Emotional Connection**: Warm, inviting atmosphere that celebrates seafood heritage

### Color Palette
- **Deep Ocean Blue** (#1e3a8a): Primary brand color, represents depth and freshness
- **Auspicious Red** (#dc2626): Accent color for CTAs and important elements
- **Gold** (#f59e0b): Luxury accent, highlights and premium touches
- **Warm White** (#fefdf8): Background and text areas
- **Charcoal** (#374151): Text and subtle elements

### Typography
- **Display Font**: "Kanit" (Google Fonts) - Modern Thai-inspired sans-serif
- **Body Font**: "Prompt" (Google Fonts) - Clean, readable Thai-supported font
- **Hierarchy**: 
  - Hero Headlines: 2.5rem (40px) bold
  - Section Headers: 1.875rem (30px) semibold
  - Body Text: 0.875rem (14px) regular
  - Captions: 0.75rem (12px) light

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth fade-in animations for sections and menu items
- **Typed.js**: Typewriter effect for hero tagline
- **Splide**: Image carousel for menu gallery
- **ECharts.js**: Data visualization for restaurant analytics (admin)
- **p5.js**: Particle effects for background ambiance

### Animation Effects
- **Hero Typewriter**: Tagline types out with gold color cycling
- **Scroll Animations**: Sections fade in as user scrolls (opacity 0.9 to 1.0)
- **Menu Hover**: Subtle scale and shadow effects on menu cards
- **Button Interactions**: Smooth color transitions and micro-animations
- **Loading States**: Skeleton screens for image loading

### Header Effect
- **Hero Background**: Custom generated restaurant interior image
- **Gradient Overlay**: Subtle blue-to-transparent gradient for text readability
- **Floating Elements**: Subtle particle effects using p5.js
- **Responsive Scaling**: Perfect fit for mobile and desktop

### Layout Principles
- **Grid System**: Tailwind CSS responsive grid
- **Mobile Navigation**: Hamburger menu with smooth slide animation
- **Card Design**: Rounded corners, subtle shadows, hover effects
- **Spacing**: Generous whitespace for premium feel
- **Touch Targets**: Minimum 44px for mobile accessibility

### Interactive Elements
- **Reservation Form**: Real-time validation with visual feedback
- **Menu Gallery**: Lightbox modal for full-screen image viewing
- **Admin Dashboard**: Drag-and-drop image upload simulation
- **Career Form**: Multi-step form with progress indicator
- **Location Map**: Clickable Google Maps integration

### Image Treatment
- **Hero Image**: Full-width, high-impact restaurant interior
- **Menu Images**: Consistent aspect ratio, professional food photography
- **Background Textures**: Subtle patterns for depth without distraction
- **Icon System**: FontAwesome icons with consistent sizing and color

### Responsive Behavior
- **Mobile First**: Designed for iPhone, scaled up for desktop
- **Touch Interactions**: Optimized for finger navigation
- **Performance**: Lazy loading for images and animations
- **Accessibility**: High contrast ratios, keyboard navigation support
